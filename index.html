CW Complete Trainer

An Android app for learning and practising Morse code — built by HB9HKN. Covers everything from individual characters (Koch method) through words, sentences, QSO simulations, and internet web article conversion.

  • Koch method character training (40 characters, LCWO order)
  • Common words, HAM abbreviations, and callsign training
  • Realistic QSO, SOTA, and POTA simulations
  • Four Quick Game modes with Speed Ladder
  • QRM / QRN interference simulation
  • Text-to-Speech confirmation overlay
  • CW web browsing — convert any web article to Morse audio
