2026-04-11 Bug Fix

Callsign Pick Game — Answer Color Feedback

After tapping an option in the Callsign Pick Game, all wrong options now turn red and the correct option turns green — matching the color scheme used in all other games. The correct-answer pause was also extended to 2 seconds to give you time to review the result before the next round begins.

v0.16.6

2026-04-08 New Feature

Callsign Pick Game

A new multiple-choice game is available under Word Training → Call Signs. Instead of judging Correct / Wrong, you hear a callsign in Morse and must select the correct one from 3 – 4 options.

How it works

  • Options are shown on screen while the Morse plays, but are disabled until audio finishes.
  • A 6-second timer starts when Morse ends — tap the correct callsign before time runs out.
  • Each foil differs from the correct callsign by exactly one Morse-confusable character (e.g. H↔S, 9↔0, K↔N) — they are designed to be the most likely decoding errors.
  • TTS is suppressed regardless of your setting — the Morse audio is the only cue.
  • The Speed Ladder applies (+0.2 WPM correct, −0.1 WPM wrong).
The number of options (3 or 4) adapts to available screen height.

v0.16.5

2026-04-07 Bug Fix

Contest QSO Accuracy Improvements

Three fixes to make contest QSO simulation more realistic:

  • ARRL DX — 2×1 US callsigns: calls like W1X and K2Y now correctly identify the district digit in position 2 and generate the appropriate state exchange.
  • CWops CWT — state/province: non-member US stations now send their state, and VE stations send their province (instead of a generic country prefix).
  • Copy-failure injection: with ~10% probability per exchange, the simulated operator asks for a repeat (? or AGN) and the other station retransmits. Adds realistic QSO copy-failure handling practice.

v0.16.4

2026-04-06 Enhancement

QRM / QRN in CW Web Browsing

QRM and QRN interference now apply to CW Web Browsing playback as well as all training modes. If you have interference or atmospheric noise enabled in Settings → Audio, it will be mixed into web article audio — letting you practise real-content copy under realistic band conditions.

v0.16.2

2026-04-04 Change

Continent Preset Buttons + Contest Abbreviation Fix

The six individual continent sliders in Call Sign Settings are now complemented by three quick-select preset buttons (Europe, North America, Asia) that set all six sliders at once. Sliders remain fully adjustable individually after a preset is applied.

Also fixed: WPX contest abbreviations now only abbreviate 0 → T and 9 → N (the standard Q-code shorthand). The earlier rule that also abbreviated 1 → A was incorrect and has been removed.

v0.16.0

2026-04-03 New Feature

HAM Abbreviations Quick Game

A new Quick Game button is now available under Word Training → HAM Abbreviations. The game follows the same Correct / Wrong format as the Word game — hear an abbreviation in Morse, decide if the text on screen matches — with a 6-second timer and Speed Ladder support. TTS spells each abbreviation letter by letter when answering.

Also in this release: game screens now keep the answer text visible from the moment Morse starts, matching the behaviour of other game modes.

v0.15.1

2026-04-03 Bug Fix

Continent Slider — Proportional Redistribution

When you drag a continent percentage slider, the remaining percentage is now redistributed proportionally across all other continents based on their current values. Previously, the surplus or deficit cascaded only to the slider directly below the one you moved, which felt arbitrary.

  • Dragging North America up reduces Europe, Asia, South America, Oceania, and Africa each by their fair share.
  • The six continent sliders always sum to exactly 100%.

v0.15.0

2026-03-22 New Feature

Call Sign Settings

A new Call Sign Settings section in QSO Simulation Settings lets you control which callsigns appear during QSO training and callsign practice.

Continent Distribution

Six linked sliders (North America, Europe, Asia, South America, Oceania, Africa) always sum to 100%. The defaults reflect the real-world HAM population: NA 69%, EU 27%, and 1% each for AS/SA/OC/AF. Adjust them to focus on a specific region — for example, set EU to 100% to practise with European callsigns exclusively.

Callsign Decoration

Two additional sliders control how many callsigns carry a country prefix (e.g. FR/3B8G/P) or a portable suffix (e.g. 3B8G/M). The remainder comes from the standard callsign pool.

SOTA activators always use a portable suffix regardless of the suffix setting.
